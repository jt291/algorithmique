\def\entete#1{%
\noindent\begin{minipage}{\textwidth}
\framebox[12.6cm][l]{Nom : \hspace*{3.85cm} Pr\'enom :
\hspace*{3cm}Groupe : {\bf \ }$\rule[-0.3cm]{0cm}{1cm}$}
\hfill
\framebox[0.7cm]{\bf \ 3 $\rule[-0.3cm]{0cm}{1cm}$}
\framebox[0.7cm]{\bf \ 2 $\rule[-0.3cm]{0cm}{1cm}$}
\framebox[0.7cm]{\bf \ 1 $\rule[-0.3cm]{0cm}{1cm}$}
\framebox[0.7cm]{\bf \ 0 $\rule[-0.3cm]{0cm}{1cm}$}
\vspace*{2mm}

{\footnotesize\sc Dur\'ee: 5'\hfill Documents, t\'el\'ephones, calculettes et ordinateurs interdits.}
\vspace*{7mm}

Pour chaque question, noircir la case 
(\begin{minipage}{2em}\centering{\colorbox{lightgray!49}{~~}}\end{minipage} 
$\rightarrow$
\begin{minipage}{2em}\centering{\colorbox{red!99}{~~}}\end{minipage})
correspondant à la bonne réponse
(il n'y a qu'une seule bonne réponse par question).

Pour une question donnée, la réponse est considérée comme fausse si
\begin{itemize}
\item aucune case n'est noircie ou
\item plusieurs cases sont noircies (même si la bonne réponse est noircie) ou
\item une seule case est noircie mais ne correspond pas à la bonne réponse.
\end{itemize}
\vspace*{1cm}

\centering{\large\bf QCM : #1}
\vspace*{1cm}

\end{minipage}}
